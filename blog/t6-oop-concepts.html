Variable Scopes

Week 6 of DBC's Remote Prepwork

07/26/2015

This week we learned more about Ruby variable scopes. In this programming language, there are five variable scopes--global, instance, class, local and constants. Before explaining the purpose of each, take a look at the list below. Each variable is distinguishable by certain characteristics detailed below.

  1. CONSTANT variables - always capitalized; usually unchanged
  2. $Global variables - marked by this symbol '$';the top level in the variable hierarchy
  3. @instance variables - marked by this symbol '@';
  4. @@class variables - marked by this symbol '@@'; a strict level in the variable hierachy
  5. local_variable - usually lowercase without any symbols in front of them; local variable
